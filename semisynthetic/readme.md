## Installing Dependencies

To install Crowd-Kit, run the following command: pip install crowd-kit.

Requries cifar-10 models and cifar data to run. Please refer to the links in the paper.

## Workflow

To reproduce numerical results in "Crowdsourcing methods on semisynthetic CIFAR-10 labels", run the following files sequentially

generated_labels/main.py
generated_labels/plot.py
generated_labels/plot_repeats.py
